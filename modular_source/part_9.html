Codex Web, GitHub App, and Copilot Enterprise

@codex mentions, asynchronous PR review, GitHub Copilot agent mode

Codex integrates into GitHub on two levels: as a GitHub App (installable on any repository) and as a native agent of GitHub Copilot Enterprise (available from February 2026 for Business/Enterprise plans).

GitHub App — @codex Mentions

Install the GitHub App from chatgpt.com/codex → Settings. Once activated, Codex monitors your repositories and reacts to mentions asynchronously.

# In a GitHub Issue comment:
@codex Analyze the code in the authentication area (src/auth/).
Implement this feature respecting the rules in AGENTS.md.
Open a PR with a feature/auth-[feature-name] branch.

# On a specific line in a PR review:
@codex this if does not handle the null case. Fix it keeping the same public API.

# On a general PR comment:
@codex review this PR following AGENTS.md.
Focus on: input validation, error handling, race conditions.

GitHub Copilot Enterprise — Codex as Native Agent

From February 2026, GitHub Copilot Business/Enterprise plans can choose Codex as an agent in the native GitHub flow. This allows Codex to assign tasks from Copilot directly to repository branches without leaving the GitHub interface.

FeatureHow to activate it
Assign Issue to CodexFrom the issue, Assignee menu → choose "Codex Agent". Codex implements and opens a PR.
Automatic PR reviewIn Copilot settings, enable "Auto-review on PR open". Codex comments on every PR within 2 minutes.
Fix from review commentComment a line with @copilot fix this. Codex generates the corrective commit in the PR branch.
