Codex Desktop & Worktrees

Multi-agent, parallel git worktrees, review, automation, computer use

In addition to the CLI, the Codex Desktop App allows you to orchestrate complex tasks visually, especially useful for Computer Use (e.g., driving the browser for scraping or E2E testing).

Workspace TypeBehaviorWhen to use it (2026 Best Practice)
LocalWorks directly in the project directory.Fast tasks, bug fixes, unit tests. Low-risk modifications.
Worktree (--worktree)Codex automatically creates an isolated Git worktree in a temporary path.Parallel feature development, massive refactoring. Prevents conflicts with your local branch and does not pollute git status.
Cloud / SandboxExecutes in a cloud container or a secure local Docker sandbox.Long tasks (e.g., DB migration), execution of untrusted code or unknown packages (prevents supply chain attacks).
💡
Tip: Use codex --worktree feature-name to let Codex develop a feature on an invisible worktree, while you continue writing code on master.
