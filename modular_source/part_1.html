Introduction: What is OpenAI Codex in 2026

OpenAI Codex is the agentic coding platform from OpenAI. It is not just simple code completion: it is a multi-surface system that can read, write, execute, and debug code autonomously, in parallel, on the cloud, and locally. The official repository is github.com/openai/codex.

Launched in preview in April 2025 and now available on all paid ChatGPT plans (Plus, Pro, Team, Enterprise), Codex is powered by the codex-1 family of models (optimized for SWE tasks) and the GPT-5.x general-purpose models.

ℹ
What changed in 2026: OpenAI has progressively unified the "Codex" branding. Dedicated models (e.g., code-davinci-002) have been deprecated. Today, the Codex CLI uses codex-mini-latest (alias for o4-mini) by default, which excels at code reasoning at a low cost. For architectural tasks, gpt-5.5 is used.
