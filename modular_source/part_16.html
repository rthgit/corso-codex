Final Capstone — Managing a Full Feature from the Desktop App

The final project demonstrating real competence.

The goal is not "to make it write code." The goal is to demonstrate that you know how to use the Desktop App as an agentic work environment: brief, plan, worktree, execution, test, browser, review, follow-up, PR.

Initial BriefWrite goal, context, constraints, done.
Approved PlanCodex proposes files, risks, tests. You correct and approve.
Dedicated WorktreeThe agent works isolated from the main branch.
ImplementationSmall changes, targeted tests, readable logs.
UI VerificationIf there is a frontend, use the integrated browser to validate real behavior.
Adversarial ReviewAsk for a critical review of the diff.
Final PRSummary, tests, residual risks, title, and description.
