## Purpose

This repository contains the definitive OpenAI Codex course in Italian and English. It is designed for software engineers, technical founders, tech leads, and enterprise teams that want to design, govern, and verify agentic workflows with Codex.

The focus is not simple code completion. The course teaches how to use Codex as an operational environment for planning, editing, testing, reviewing, automating, and orchestrating subagents under controlled policies.

## Course Files

- [Italian course](openai_codex_corso_definitivo_it.html)
- [English course](openai_codex_corso_definitivo_en.html)

Open the HTML file directly in a modern browser. No build step, local server, or dependencies are required.


## Models Updated to April 29, 2026

The course uses the updated model hierarchy:

- `GPT-5.5`: frontier model for Codex, long-running tasks, architecture, Computer Use, and complex agentic coding.
- `GPT-5.3-Codex`: specialized Codex/API model for agentic coding workflows.
- `GPT-5.4`, `GPT-5.4-mini`, `GPT-5.4-nano`: alternatives for cost, latency, and narrower tasks.
