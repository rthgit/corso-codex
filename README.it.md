## Scopo

Questo repository contiene il corso definitivo su OpenAI Codex in italiano e inglese. Il corso è pensato per software engineer, founder tecnici, tech lead e team enterprise che vogliono imparare a progettare, governare e verificare workflow agentici con Codex.

Il focus non è il semplice completamento di codice: il corso insegna come usare Codex come ambiente operativo per pianificare, modificare, testare, revisionare, automatizzare e orchestrare subagenti in modo controllato.

## File del Corso

- [Corso italiano](openai_codex_corso_definitivo_it.html)
- [English course](openai_codex_corso_definitivo_en.html)

Apri il file HTML direttamente in un browser moderno. Non servono build, server locale o dipendenze.

```powershell
start openai_codex_corso_definitivo_it.html
```

## Modelli Aggiornati al 29 Aprile 2026

Il corso usa la gerarchia aggiornata:

- `GPT-5.5`: modello frontier per Codex, task lunghi, architettura, Computer Use e coding agentico complesso.
- `GPT-5.3-Codex`: modello specializzato Codex/API per workflow di coding agentico.
- `GPT-5.4`, `GPT-5.4-mini`, `GPT-5.4-nano`: alternative per costo, latenza e task più circoscritti.
